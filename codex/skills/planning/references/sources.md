# Primary references

Accessed 2026-07-13. Use these as selectable planning lenses; repository constraints and the requested outcome remain authoritative.

| Source | Publisher/owner and currentness | Governed topic |
| --- | --- | --- |
| [ISO 21502:2020](https://www.iso.org/standard/74947.html) | ISO, published 2020 | General project-management practices, stakeholders, planning, risk, change, and delivery context |
| [NASA Systems Engineering Handbook](https://www.nasa.gov/reference/systems-engineering-handbook/) | NASA/SP-2016-6105 Rev. 2; official page maintained | Technical planning, requirements, interfaces, decision analysis, risk, verification, validation, and lifecycle reviews |
| [SEI Architecture Tradeoff Analysis Method](https://www.sei.cmu.edu/library/architecture-tradeoff-analysis-method-atam/) | Carnegie Mellon SEI; method page updated 2026-05-11 | Stakeholder scenarios and architecture risk, sensitivity, and tradeoff analysis for consequential decisions |
| [Architecture Decision Records](https://adr.github.io/) | Maintained ADR community hub | Durable decision context, status, alternatives, consequences, and supersession |
| [DORA: working in small batches](https://dora.dev/capabilities/working-in-small-batches/) | Google Cloud DORA, maintained research synthesis | Small, independently testable, deployable increments and faster feedback |
| [Scrum Guide](https://scrumguides.org/scrum-guide.html) | Ken Schwaber and Jeff Sutherland, current official November 2020 version | Scrum-specific accountabilities, events, commitments, and empiricism; use only when the team uses Scrum |
| [arc42 overview](https://arc42.org/overview) | Maintained official architecture template | Tailorable architecture-planning and documentation topics without mandatory ceremony |
| [C4 model](https://c4model.com/) | Maintained official model site | Context and container views when the plan needs durable boundary communication |

The sources do not justify a formal ceremony by themselves. Match planning depth to uncertainty, reversibility, coordination, and impact.
