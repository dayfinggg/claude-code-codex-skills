# Authoritative sources

Standards define semantics; they do not prove that a target runtime implements them. Proposal repositories can lag the integrated living specification. Verify the repository's target browsers/runtimes and installed tooling before using version-sensitive features. All sources were inspected and accessed 2026-07-13.
